Set Up

This is how to set up the Rubik's Cube project into Eclipse

Import cp411_project_Iarr520_Barb8230.zip into Eclipse, use the "Existing Project into Workspace" option and selecting the projects Zip.

From there you can build the project by right clicking the project and selecting "Build Project" the project should build without issue.

Once the project is built you are ready to run the project. To do this right click the proejct again and hover the "Run As" option

select the "C++ application" option from the sub menu and the project should run

The Game

The point of the game is to solve the rubik's cube, the one at the start is how it will looked solved,

you will know its been solved as an audio clip of trumpets will play.

This game is primarily played through the menu and keybaord inputs.

The Menu

By Right clicking on the game space the menu will appear. From there your options are:

  1. Reset - reset the cube back to the beginning
  2. Scramble - Shuffle the faces to try and solve
  3. Shading - Different Textures for the cube
  4. Camera - Rotate the Camera in the X, Y or Z axis
  5. Rotate - Rotate the Desired face, the user can select: Top, Bottom, Right, Left, Front, Back
  6. Quit - Exit the Game

We will now take a deeper look into each menu

Reset

This will reset the game, the player will not hear the winning audio because they gave up and reset

Scramble

This scrambles the cube and is the easiest way to try and solve the cube

Shading

There are three shading options for the rubik's cube

Constant - Which is the default, the tile is one solid color

Wire - There are wire X's on each tile and a border all in the color of the tile, vector graphics inspired

Monkey - The Monkey fRom CP411 assignments makes a return one for each tile with the tile color providing the differentiation of where each monkey belongs

Camera

The camera works by selectign one of the Three sub menu options

X - Rotate along the X

Y - Rotate along the Y

Z - Rotate along the Z

The rotation works based on HBF implementation, if the mouse is to the left of where you clicked you rotate counter clockwise, the right is clockwise

Rotate

The Rotate is how we move the faces and solve the unsolved cubes

The six faces are:

  1. Top - White - Z Key
  2. Bottom - Yellow - X Key
  3. Left - Red - C Key
  4. Right - Orange - V Key
  5. Front - Blue - B Key
  6. Back - Green - N Key

Using a series of these 6 primary motions you can solve any Rubik's Cube

Quit

This is how you leave the game, you have free will, you can use it

Keyboard inputs and face rotates

As mentioned in the "Rotate" Section, there are keys assosiated to the faces,

by clicking the coresponding key that face will rotate.

Another way to Rotate is by using the R key, this will rotate the last selected face.
